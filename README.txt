Sketcher Tab is a procedural drawing program. It contains 7 brushes, undo/redo copability. You can share images by mail,
Dropbox, Picasa, etc. Interface optimized for Android tablets. Minimal system requirements is Android 3.0.

See example pictures on the Market page
https://market.android.com/details?id=org.sketchertab

The idea and implementation created by Mr.doob in Harmony.
http://mrdoob.com/projects/harmony/

First java implementation was made by Wargoth.
https://github.com/wargoth/Sketcher

Sketcher Tab uses sources of Sketcher Lite, thanks to Wargoth.
